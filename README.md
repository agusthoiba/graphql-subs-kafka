# graphql-subs-kafka
Graphql subscription Kafka

Graphql subscription with kafka example

## System Requirement
nodejs 14
mongodb 4.2
kafka 7.0.1-ce
 
## Installation
```bash
git clone https://github.com/agusthoiba/graphql-subs-kafka.git
npm i
npx tsc
```

## Usage
```bash
node src/index.js
```
